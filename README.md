# 🎵 Music Player

A simple web-based music player built using HTML, CSS, and JavaScript.

## 🚀 Features
- Play, pause, and stop songs
- Display lyrics
- Show background video
- Dynamic song list

## 🧠 How It Works
The player uses JavaScript functions to load songs, display lyrics, and sync the visuals.
